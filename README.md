# bd_pz5
**ПЗ5_БД_234_Вольников_Шиневский - Приложение для поиска калорийности продуктов по названию или коду**

Простое desktop-приложение для поиска и анализа пищевой ценности продуктов через Open Food Facts API.

**Основной функционал**
- **Поиск по названию** - найти продукты по ключевым словам (яблоко, хлеб, молоко)
- **Поиск по штрихкоду** - точный поиск по EAN-13 коду (13 цифр)
- **Таблица результатов** - просмотр списка найденных продуктов с основной информацией
- **Детальная информация** - полные данные о выбранном продукте
- **Пищевая ценность** - калории, белки, жиры, углеводы на 100г

**Базовое использование:**\
  1. Запустите приложение\
  2. Введите запрос в поле поиска:\
  3. Название продукта: яблоко, шоколад, хлеб\
  4. Штрихкод: 4607065378327, 5449000000996\
  5. Нажмите "Найти" или клавишу Enter\
  6. Дважды щелкните по продукту в таблице для детальной информации

**Интерфейс**\
Вкладка "Результаты":\
Название - наименование продукта\
Бренд - производитель\
Штрихкод - уникальный идентификатор\
Ккал - калории на 100 грамм\
Белки - содержание белков (г/100г)\
Жиры - содержание жиров (г/100г)\
Углеводы - содержание углеводов (г/100г)

**Вкладка "Детали":**\
Основная информация - название, бренд, штрихкод, упаковка\
Пищевая ценность - подробный состав на 100 грамм
